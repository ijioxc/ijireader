# 上架到 GitHub Pages 完整指引

## 最終目標資料夾結構

```
你的 repo/
├── index.html          ← 書架頁（從本方案取得）
├── reader.html         ← 閱讀器（從本方案取得）
└── books/
    ├── shadow_slave/
    │   ├── manifest.json
    │   └── chapters/
    │       ├── 0001.json
    │       ├── 0002.json
    │       └── ...
    └── lotm/
        ├── manifest.json
        └── chapters/
            ├── 0001.json
            └── ...
```

---

## Step 1 — 建立 GitHub 帳號與 Repo

1. 前往 https://github.com，登入或建立帳號
2. 點右上角 **+** → **New repository**
3. Repository name：`bilingual-reader`（任意名稱）
4. 選 **Public**（GitHub Pages 免費方案只支援 Public）
5. 勾選 **Add a README file**
6. 按 **Create repository**

---

## Step 2 — 上傳檔案

### 方法 A：直接用網頁上傳（不需要懂 Git）

1. 進入你的 repo 頁面
2. 點 **Add file** → **Upload files**
3. 先上傳 `index.html` 和 `reader.html`
4. 建立資料夾的方式：
   - 點 **Add file** → **Create new file**
   - 在 filename 欄位輸入：`books/shadow_slave/manifest.json`
   - 貼上 manifest.json 的內容
   - 按 **Commit new file**
5. 重複以上步驟，把每個 `chapters/0001.json` 等都上傳進去

> ⚠️ 章節多的話建議用方法 B

### 方法 B：用 Git 上傳（推薦，一次搞定）

1. 安裝 Git：https://git-scm.com/downloads
2. 在你電腦上建好完整資料夾結構（見上方）
3. 打開終端機，進入你的資料夾：

```bash
cd /你的資料夾路徑

# 初始化 git
git init
git add .
git commit -m "first commit"

# 連接到 GitHub（把 YOUR_USERNAME 換成你的帳號名）
git remote add origin https://github.com/YOUR_USERNAME/bilingual-reader.git
git branch -M main
git push -u origin main
```

---

## Step 3 — 開啟 GitHub Pages

1. 進入 repo → 點上方 **Settings** 標籤
2. 左側選單找到 **Pages**
3. 在 **Source** 下選 **Deploy from a branch**
4. Branch 選 **main**，資料夾選 **/ (root)**
5. 按 **Save**
6. 等 1-2 分鐘，頁面會顯示你的網址：

```
https://YOUR_USERNAME.github.io/bilingual-reader/
```

---

## Step 4 — 設定 DeepL API Key

1. 前往 https://www.deepl.com/pro-api 申請免費帳號
2. 在 API Keys 頁面取得你的 key（結尾會是 `:fx`）
3. 開啟你的網站 → 點右上角 ⚙ 設定
4. 在「DeepL 翻譯 API」欄位貼入你的 key → 按儲存
5. Key 儲存在你的裝置 localStorage，不會上傳任何地方

---

## Step 5 — 手機/平板存取

直接用瀏覽器打開你的 GitHub Pages 網址即可，支援所有裝置。

### 加入主畫面（像 App）

**iPhone / iPad：**
1. 用 Safari 開啟你的網址
2. 點底部分享鍵 □↑
3. 選「加入主畫面」→ 加入
4. 之後直接從主畫面開啟，全螢幕顯示

**Android：**
1. 用 Chrome 開啟你的網址
2. 點右上角 ⋮ → 加到主畫面

---

## 新增書籍

1. 在 `books/` 下新增資料夾，例如 `books/new_book/`
2. 放入 `manifest.json` 和 `chapters/` 資料夾
3. 在 `index.html` 第 44-55 行的 `BOOK_LIST` 陣列加入：

```js
{
  id: 'new_book',
  title: '書名',
  desc: '簡介...',
  manifest: 'books/new_book/manifest.json'
}
```

4. 推 git 或上傳到 GitHub，完成

## 常見問題

| 問題 | 原因 | 解法 |
|------|------|------|
| 章節載入失敗 | json 路徑不對 | 確認 manifest.json 裡的 `url` 欄位與實際路徑一致 |
| DeepL 回傳 403 | API Key 錯誤 | 重新貼入 key，確認結尾有 `:fx` |
| 手機上語音不播 | 瀏覽器限制 | 先手動點一次段落觸發播放，iOS 需要用戶互動才能啟動音訊 |
| GitHub Pages 顯示 404 | Pages 還在部署 | 等 2-3 分鐘再重整 |
| 翻譯快取滿了 | localStorage 上限 5MB | 在設定中清除翻譯快取 |
